# HyperNet 安全对齐迁移

本研究实现一种基于超网络（Hypernetwork）的安全对齐迁移框架，
用于将大模型（LLM）中的安全对齐能力结构化地迁移到小模型（SLM）。

核心思想：
- 不直接使用大模型参数；
- 从大模型中学习“功能基向量”（Functional Basis），刻画参数的作用模式；
- 使用跨模型注意力机制（Q=小模型激活，K=大模型激活，V=功能基）；
- 注意力输出作为条件信号，与层 embedding 拼接后输入超网；
- 超网生成对小模型参数的调制信号（如 scaler / delta）。

该方法强调：
- 参数级而非纯行为级对齐
- 可控、低秩、可扩展
- 适用于安全对齐与模型压缩研究
